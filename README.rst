``h5bp_jinja``
==============


A small script that converts an `HTML5 Boilerplate`_ template into a `Jinja2`_
template.

For usage instructions, consult ``--help``.

.. _HTML5 Boilerplate: http://html5boilerplate.com/
.. _Jinja2: http://jinja.pocoo.org/
